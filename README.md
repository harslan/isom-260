# ISOM 260: AI for Business - Session 2

## 🧠 How AI Actually Works: From Neurons to LLMs

An interactive lecture website for teaching the fundamentals of neural networks and Large Language Models to business students.

**Course:** ISOM 260 - AI for Business  
**Institution:** Suffolk University, Sawyer Business School  
**Instructor:** Prof. Hasan Arslan

---

## 🌐 Live Website

Visit: `https://[your-username].github.io/isom260-session2/`

---

## 📚 Session Overview

This 150-minute session covers:

1. **The Big Picture** (25 min) - Visual introduction to neural networks
2. **Interactive Exploration** (40 min) - Hands-on with TensorFlow Playground & LLM Visualization
3. **How LLMs Are Built** (35 min) - Pre-training, tokenization, and post-training
4. **Business Applications & Limitations** (30 min) - Real-world value and "jagged intelligence"
5. **Build Challenge** (25 min) - Create your own classifier with Teachable Machine
6. **Wrap-Up** (10 min) - Key takeaways and homework

## 🚀 Deploying to GitHub Pages

1. Create a new repository on GitHub
2. Upload `index.html` to the repository
3. Go to **Settings** → **Pages**
4. Under "Source", select **main branch**
5. Save and wait a few minutes
6. Your site will be live at `https://[username].github.io/[repo-name]/`

---

## 💻 Local Development

Simply open `index.html` in any modern web browser. No build step or server required.

## 📝 Learning Objectives

By the end of this session, students will be able to:

1. Explain what a neural network is and visualize how it "learns"
2. Understand how LLMs (like ChatGPT) work at a conceptual level
3. Connect neural network concepts to business applications
4. Build intuition through hands-on interactive tools
