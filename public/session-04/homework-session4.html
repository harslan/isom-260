Overview

In class you built a multi-tool AI agent that can reason about which tools to use, when to use them, and how to combine results into a coherent answer. This homework asks you to go further: design and build your own tool, stress-test your agent, and reflect on what you learned.

You are encouraged to use AI tools (ChatGPT, Claude, Gemini) to help you understand code and complete tasks. That’s a core skill in this course.

2 Build a Custom Tool

Build — 40 pts

Design and implement your own tool, then test it with 3 queries

In Part 5 of the notebook, use the template to create a 4th tool. You need to define:

  • Tool name — a descriptive identifier (e.g., "calculate_tip_split")
  • Description — 1–2 sentences telling Gemini exactly when to use your tool
  • Parameters — the inputs your tool needs (name, type, description for each)
  • Function body — Python code that takes the parameters and returns a useful string

You may use simulated/hardcoded data (like the stock tool does). Creativity counts.

Test your tool with 3 queries:

  • Query 1: A straightforward use of your tool
  • Query 2: A different scenario or input
  • Query 3 (multi-tool): A question that requires your tool plus at least one existing tool (calculator, stock, or company info)

Deliverable: Tool definition, function, and 3 test runs with visible output.

3 Write Reflections

Write — 20 pts

Answer these three questions in text cells (3–5 sentences each)

  1. Tool descriptions matter. Change your tool’s description to something vague (e.g., “A tool that does stuff”). Run the same query. What happens? Why does description quality matter for an AI agent? (Change it back after.)
  2. Agent vs. chatbot. Explain the difference between a chatbot and an AI agent to someone who has never taken this class. Use a concrete business example.
  3. Your agent product. If you could build an AI agent for a real company or organization, what would it do? What 3 tools would it need? Who would use it and why?

Deliverable: Written answers in text cell(s) at the bottom of your notebook.

4 Bonus: Break Your Agent

Stretch — 10 bonus pts

Find the limits of your agent

Try to make your agent produce a wrong answer or behave unexpectedly. Ideas:

  • Ask something that requires a tool you haven’t provided
  • Ask something ambiguous that could match multiple tools
  • Give conflicting instructions in one question

Deliverable: 1–2 test runs plus a brief explanation of what happened and why (in a text cell).
